# Scent-ID Protocol v1.0
## 數位感官全球標準化協議 (The Digital Standard for Olfactory Perception)

Scent-ID (SID) 是一套將嗅覺數位化的通用框架。我們透過 AI 建模將主觀的氣味描述轉換為精準的 **256 維化學向量**，實現氣味在數位世界的精準傳遞、搜尋與重現。

### 核心架構 (Core Architecture)
- **維度定義：** 256 種基礎化學分子座標 (S-001 至 S-256)。
- **技術格式：** 基於 SHA-256 加密索引與數位簽章。
- **管理機構：** Scent-ID Foundation (全球氣味授權局)。
